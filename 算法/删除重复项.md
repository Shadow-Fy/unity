# 删除重复项



通过两个指针移动控制是否有重复



```c
int removeDuplicates(int* nums, int numsSize)
{
    int flagA = 0, flagB = 0;
    for(;flagB < numsSize; flagB++)
    {
        if(nums[flagA] != nums[flagB])
        {
            flagA++;
            nums[flagA] = nums[flagB];
        }
    }
    return flagA+1;
}
```
